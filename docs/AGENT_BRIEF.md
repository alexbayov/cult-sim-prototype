# Agent brief

Этот файл нужен, чтобы следующий агент быстро понял проект и не сбился с курса.

## Что это за проект

Карточная web-игра о том, как обычная группа/марафон/сообщество может постепенно собираться в деструктивную систему.

Игрок не получает прямую лекцию. Он видит:

- нейтральную внешнюю упаковку практик;
- состояние участников;
- групповые метрики;
- комбо-паттерны;
- красные флаги;
- финальный разбор.

## Текущий стек

- React
- TypeScript
- Vite
- CSS
- пока без backend
- пока без внешней базы

## Репозиторий

- GitHub: `https://github.com/alexbayov/cult-sim-prototype`
- Основная ветка: `main`

```bash

```

```bash
npm run build
npm run lint
```

## Главная продуктовая рамка

Это не «симулятор как эффективно ломать людей».

Это модель, где:

- практики часто выглядят нормально снаружи;
- вред возникает из сочетаний, контекста и накопления;
- игра показывает цену решений через людей, а не через морализаторство;
- финальный разбор переводит события партии в термины психпросвещения.

## Семантика карт

Не писать на surface-слое:

- «лидер всегда прав»;
- «контроль информации»;
- «стыд за пропуск»;
- «манипуляция».

Писать как это звучало бы в реальной упаковке:

- «общая позиция команды»;
- «единый ритм потока»;
- «чистая среда фокуса»;
- «разбор внутренних стопоров»;
- «окно поддержки».

А реальная логика должна быть в:

- `tags`;
- `effects`;
- `redFlags`;
- `debriefTags`.

## Безопасность контента

Можно рассматривать:

- инфоцыганство;
- псевдопсихологию;
- эзотерику;
- скам;
- конспирологию;
- радикализацию;
- криминальные вовлечения.

Но нельзя превращать это в инструкцию:

- не давать конкретных каналов, процедур, скриптов вербовки;
- не описывать способы самоповреждения;
- не давать operational детали преступлений;
- тяжёлые темы показывать через красные флаги и последствия.

## Definition of done для любого PR

- Изменение маленькое и понятное.
- README/docs обновлены, если поменялся процесс.
- `npm run build` проходит.
- `npm run lint` проходит.
- Если добавлен контент, он соответствует `docs/CONTENT_MODEL.md`.
- Если затронуты чувствительные темы, нет инструктивных деталей.
